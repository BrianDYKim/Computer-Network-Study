1. 컴퓨터 네트워크와 인터넷(1주)

6. 러닝 HTTP 2.0 4장 5장
7. 러닝 HTTP 2.0 6장 9장

- 컴퓨터 네트워크 하양식 접근 Chapter 4까지 진행
- 러닝 HTTP 2.0 Chapter 7, 8 제외하고 1 ~ 9까지 진행
- 매주 같은 Chapter를 공부하고 한명이 발표를 맡아서 진행하고 다른 스터디원들과 의견 공유
